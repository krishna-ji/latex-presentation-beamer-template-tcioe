# Images Directory

Place all your presentation images in this directory.

## Recommended Images

- `ioe-logo.png` - IOE/Thapathali Campus logo
- Architecture diagrams
- Performance graphs
- Gantt charts
- System screenshots
- Any other visual assets

## Usage

To include an image in your slides, use:

```latex
\includegraphics[width=0.8\textwidth]{src/images/your-image.png}
```

Make sure to reference images with the full path from the main document root.
